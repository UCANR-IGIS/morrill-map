Summary

This notebook examines the amount of land that was given out in California under the 1862 Morrill Act, to support the establishment of Land Grant Universities throughout the US. The number of parcels and acreage is totaled for each state which received land under the Morril Act (i.e., California), or whose land scrip was redeemed in California (everyone else). For more information on the Morrill Act, and the role it played in establishing public universities in the US, see Wikipedia.

The data source for this exercise is a polygon layer of parcels that were transferred from the Federal government to public and private (mostly private) entities. For details see:

  • Lee, R. 2020. How we investigated the land-grant university system. High Country News.
  • Parcel layer: https://github.com/HCNData/landgrabu-data
  • Processing script for California: https://github.com/UCANR-IGIS/morrill-map/blob/master/hcn-parcels-prep.R
  • Interactive web map: http://bit.ly/ca-morrill-map

3. Number of parcels and acreage by State

The majority of land in California transferred from Federal to private ownership under the Morrill Act actually went to benefit public universities in other States. States like New York received a large allotment of land (which was distributed according to population), but

state_totals_tbl <- hcnparc_ca_sf %>% 
  st_drop_geometry() %>% 
  group_by(LG_State) %>% 
  summarise(Num_Parcels = n(), Total_acres = sum(Acres))
  
DT::datatable(state_totals_tbl, 
          class = 'row-border stripe hover compact', 
          colnames = c("State", "Number Parcels", "Total Area (acres)"),
          width = 400,
          rownames = F, 
          options = list(autoWidth = TRUE,
                         paging = FALSE,
                         searching = FALSE,
                         info = FALSE)) %>% 
  formatRound(columns = "Total_acres", digits = 0) 
ggplot(data = state_totals_tbl, 
       aes(x = Total_acres, y = reorder(LG_State, Total_acres))) + 
  geom_bar(stat="identity") +
  xlab("Acres") +
  ylab("State Proceeds Went To") +
  ggtitle("Parcels in California Transferred Under the 1862 Morrill Act") +
  scale_x_continuous(labels = scales::comma)
